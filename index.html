Referência de tabela HTML

Etiquetas de tabela

Marcação Nome Descrição
table Table O elemento wrapper para todas as tabelas HTML.
thead Table Head O conjunto de linhas que definem os cabeçalhos das colunas em uma tabela.
tbody Table Body O conjunto de linhas que contém dados reais da tabela.
tr Table Row O contêiner de linha da tabela.
td Table Data O contêiner de linha da tabela.
tfoot Table Foot O conjunto de linhas que definem o rodapé em uma tabela.

Atributos da tabela

Atributo Nome Descrição
colspan Extensão da coluna Define quantas colunas um elemento td deve abranger.
rowspan Extensão de linha Define quantas linhas um elemento td deve abranger.
